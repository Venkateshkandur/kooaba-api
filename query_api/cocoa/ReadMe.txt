Build & Run
===========

This Cocoa sample project contains both targets for iPhone OS and Mac OS X (Foundation Tool).

The Base SDK of the project is set to Mac OS X 10.5, the Base SDK of the iPhone Target is set to iPhone OS 3.0 (Device). 

If you want to build for the iPhone Simulator, be sure to set the Active SDK to "iPhone Simulator 3.0" and the active target to "kooabaQueryApp".


Classes
=======

The common source files (for iPhone OS and Mac OS X) are located in the "Kooaba" and "Library" groups, the target-specific ones in "Mac" and "iPhone" respectively.


Enjoy!
